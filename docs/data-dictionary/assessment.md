# Kamus Data: Assessment

Assessment dapat memuat:

- kondisi fisik;
- kondisi psikologis;
- kondisi sosial/ekonomi/spiritual;
- kondisi hukum;
- upaya sebelumnya;
- faktor pendukung;
- hambatan;
- kebutuhan atau harapan klien;
- informasi lain yang relevan.

Gunakan `source_id` stabil dan `assessed_at` dalam ISO 8601. Isi asesmen diklasifikasikan sangat rahasia.
