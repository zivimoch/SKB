# Kamus Data: Intervention

Intervensi terdiri atas cycle, activity/agenda, dan report/todo petugas.

Status report:

- `pending`
- `accepted`
- `rejected`
- `done`

Setiap agenda dan report wajib memiliki `source_id`. Actor yang mengubah report harus memiliki hak terhadap kasus dan menjadi pemilik todo bila aturan layanan mensyaratkannya.
