# Tata Kelola Pertukaran Data

Sebelum production, para pihak wajib menyepakati:

- tujuan dan dasar hukum pemrosesan;
- data controller/data processor;
- kategori dan klasifikasi data;
- role serta scope akses;
- retensi, koreksi, penghapusan, dan pemusnahan;
- lokasi penyimpanan dan subprocessor;
- SLA dan maintenance window;
- pelaporan insiden dan batas waktu notifikasi;
- audit, monitoring, serta hak pemeriksaan;
- prosedur penghentian integrasi.

Dokumen ini adalah checklist teknis dan bukan pengganti kajian hukum instansi.
