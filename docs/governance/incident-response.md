# Incident Response Integrasi

Saat insiden:

1. hentikan atau batasi integration client terdampak;
2. rotasi secret/sertifikat;
3. preservasi audit log dan request ID;
4. identifikasi data, actor, waktu, dan sistem terdampak;
5. koordinasikan PIC keamanan dan perlindungan data;
6. lakukan notifikasi sesuai perjanjian dan regulasi;
7. pulihkan layanan secara bertahap;
8. dokumentasikan akar masalah dan tindakan pencegahan.

Jangan menyalin payload rahasia ke tiket insiden umum.
