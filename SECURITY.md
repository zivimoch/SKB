# Security Policy

Jangan melaporkan kerentanan melalui public issue. Hubungi tim keamanan SKB melalui kanal resmi instansi.

Laporan minimal memuat versi, endpoint, dampak, langkah reproduksi aman, dan request ID. Jangan menyertakan data kasus nyata, secret, token, private key, atau dump database.

Kontrol baseline:

- least privilege dan scope;
- object-level authorization;
- HMAC anti-replay dan idempotency;
- encryption in transit dan at rest;
- audit aplikasi serta actor;
- dependency scanning, SAST, DAST, dan penetration test;
- secret rotation dan incident response.
